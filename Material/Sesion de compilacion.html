Compilamos y ejecutamos un programa muy simple

oso@osobook ~/sl $ cat main.c
#include <stdio.h>
main()
{
	printf("Hola, Mundo\n");
}
oso@osobook ~/sl $ gcc -o main main.c
oso@osobook ~/sl $ ./main
Hola, Mundo

Reutilizamos la biblioteca compartida

oso@osobook ~/sl $ cat otraprueba.c
#include <stdio.h>
main() {
	hola();
	hola();
	printf("Nada más\n");
}
oso@osobook ~/sl $ gcc -o otraprueba otraprueba.c -L. -lcarteles
oso@osobook ~/sl $ ./otraprueba
Hola
Hola
Nada más
